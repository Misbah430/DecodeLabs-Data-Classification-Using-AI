

##  Data Classification Using AI

### 📖 Overview

This project demonstrates a complete **Machine Learning Classification** workflow using an **E-commerce Sales Dataset**.

The objective is to predict the **Order Status** of customer orders using a **Decision Tree Classifier**.

---

## 🎯 Objectives

- Load and understand the dataset
- Perform data preprocessing
- Split data into training and testing sets
- Train a Decision Tree Classification model
- Evaluate model performance
- Predict Order Status for new data
- Visualize the Decision Tree

---

## 📊 Dataset Features

- OrderID
- Date
- CustomerID
- Product
- Quantity
- UnitPrice
- ShippingAddress
- PaymentMethod
- OrderStatus (Target Variable)
- TrackingNumber
- ItemsInCart
- CouponCode
- ReferralSource
- TotalPrice
